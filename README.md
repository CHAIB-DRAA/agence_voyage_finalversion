# Omra/Hajj Quote App (Expo React Native)

Minimal Expo React Native app that includes screens:
- Home (Menu)
- Add / Edit Quote
- List Quotes
- Quote Details

Data persistence uses AsyncStorage.

## Setup

1. Install Node.js and npm
2. Install Expo CLI: `npm install -g expo-cli`
3. In project folder run:
   ```
   npm install
   npm start
   ```
4. Use Expo Go on mobile or run on emulator.

This is a starter template. Improve navigation, styling and validation for production.
